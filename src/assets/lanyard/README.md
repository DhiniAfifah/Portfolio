# Lanyard Assets

Please add the following files to this directory:

1. **card.glb** - The 3D model file for the card
2. **lanyard.png** - The texture file for the lanyard band

You can download these files from the original repository or create your own:

- Edit card.glb using: https://modelviewer.dev/editor/
- Edit lanyard.png in any image editor

The component will not work without these files.
